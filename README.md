# Mukut Singh's Portfolio

A modern, dynamic, and responsive terminal-themed portfolio website.

## Features
- **Terminal interface** with custom interactive commands (`help`, `about`, `education`, `skills`, `projects`, `contact`, `clear`, etc.)
- **Interactive DOS-style interface** for an retro-future tech vibe
- **Timeline visualizer** for achievements, extracurriculars, and coursework
- **Smooth light/dark mode transition** with tailored, eye-catching color palettes and dynamic connection node animation
- **Responsive design** optimized for both mobile and desktop screens

## Tech Stack
- HTML5
- CSS3 (Vanilla custom styling)
- JavaScript (Vanilla DOM interactions and animations)

## Local Development
To run this project locally:
1. Clone the repository.
2. Open `index.html` in a web browser or run a simple local server:
   ```bash
   python -m http.server 8000
   ```
3. Visit `http://localhost:8000` in your browser.
